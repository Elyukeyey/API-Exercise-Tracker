Exercise tracker API

Create a New User

You can use the POST /api/exercise/new-user endpoint, passing in a "username" to create a new user, which returns the object { username: "test_user_1954", _id: "9ysxuu"}. You can then use the ID to track the exercises.

Add Exercises to the User

Use the POST /api/exercise/add endpoint, passing in userId, description, duration (Number) and date (in yyyy-mm-dd format) to add the exercises.

Test add exercises

Get users exercise log

Use the GET /api/exercise/log?{userId}[&from][&to][&limit] endpoint to retrieve the exercises.

userId is required, the rest of the fields are optional.
